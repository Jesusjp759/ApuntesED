## **Herramientas de desarrollo**

### 1) Ejecuta el programa "Hola mundo" en los siguientes lenguajes: 

   - bash
   - python
   - php
   - javascript (nodejs)
   - c
   - c++
   - java
   - ruby
   - go
   - rust
   - lisp
   - ensamblador (nasm)
